About

This app provides a graphical representation of the Graham Scan algorithm for finding the convex hull of a set of points

Instructions

Click the canvas to add a new point. Once you have enough points the convex hull will be drawn. You can use the buttons above to see each step of the algorithm or to clear the canvas
